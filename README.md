# Libft
Reimplementing specific libc functions and creating custom C functions for string, memory, and linked list manipulation.

## Functions from libc

### String Manipulation Functions

* `isalpha`
* `isdigit`
* `isalnum`
* `isascii`
* `isprint`
* `strlen`
* `strchr`
* `strrchr`
* `strncmp`
* `strnstr`
* `strlcpy`
* `strlcat`
* `atoi`
* `strdup`

### Memory Manipulation Functions

* `memset`
* `bzero`
* `memcpy`
* `memmove`
* `memchr`
* `memcmp`
* `calloc`
